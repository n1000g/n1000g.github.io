# Perf

| 场景 | 命令        | 解释 |
| ---- | ----------- | ---- |
| 采样 | perf record | -g   |
| 转换 | perf script |      |
| 查看 | perf report |      |

`perf list` 显示所有当前系统支持的性能事件

- **Hardware events**: CPU相关的硬件事件，例如指令计数、缓存命中等。
- **Software events**: 软件层面的事件，例如上下文切换、页面错误等。
- **Tracepoints**: 内核中的特定追踪点，例如 `syscalls` 相关的事件。
- **PMU events**: 与特定的硬件计数器（如内存控制器）相关的事件。
- **Power events**: 与电源管理相关的事件。
